Frontend Development using React

Welcome to the Frontend Development using React course! In this program, you will learn the fundamentals of frontend development and how to build web applications using the React library.
The course is best suitable for people with familiarity with web development.

Nên chọn Tailwind CSS hay Bootstrap?

  • Chọn Tailwind CSS: Nếu bạn muốn linh hoạt và thiết kế độc đáo từ đầu.
  • Chọn Bootstrap: Nếu bạn muốn có giao diện sẵn và triển khai nhanh chóng, hoặc làm việc trên những dự án quen thuộc cần sự nhất quán.

    • Nên học Framework nào trước?

  • React: Dành cho giao diện người dùng hiện đại.
  • Vue.js hoặc Angular: Tùy thuộc vào nhu cầu và môi trường làm việc.
  • Lưu ý học cơ bản JavaScript trước khi học Framework.

    • Front-End Developer có cần biết về Back-End không?

    Không bắt buộc, nhưng nên hiểu cơ bản:

  • Cách giao tiếp qua API (REST, GraphQL).
  • Kiến thức về cơ sở dữ liệu cơ bản (MySQL, MongoDB) có thể hữu ích.

    • Có cần biết thiết kế không?

    Không cần giỏi thiết kế, nhưng:

  • Hiểu cơ bản về UX/UI sẽ giúp bạn làm việc tốt hơn với Designer.
  • Học các công cụ như Figma hoặc Sketch để hiểu ý tưởng thiết kế.

    • Làm thế nào để luyện tập hiệu quả?

  • Tạo các dự án nhỏ (to-do list, trang portfolio, ứng dụng thời tiết).
  • Tham gia các thử thách trên các trang như Frontend Mentor hoặc CodePen.
  • Thực hành thường xuyên và cập nhật các xu hướng mới.
